# Monitor_Risco_Chuva
Aplicação cliente-servidor que realiza integração com APIs públicas para conversão de cidade em coordenadas geográficas e consulta de precipitação atual, aplicando regras de negócio para classificação de risco (baixo, médio, alto).  Implementa arquitetura simples de backend REST com FastAPI e frontend estático consumindo endpoint via Fetch API.
